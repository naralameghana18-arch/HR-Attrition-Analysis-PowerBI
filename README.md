# 📊 HR Attrition Analysis Dashboard (Power BI)

## 📌 Project Overview
This project focuses on analyzing employee attrition patterns to identify key factors influencing employee turnover. The dashboard provides actionable insights to help organizations improve employee retention strategies.

---

## 🎯 Objectives
- Analyze employee attrition trends
- Identify factors affecting employee turnover
- Provide insights for improving retention

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query

---

## 📂 Dataset Description
The dataset includes employee-related attributes such as:
- Demographics (Age, Gender, Education)
- Job details (Department, Role, Job Level)
- Compensation (Monthly Income, Salary Hike)
- Work conditions (Work-Life Balance, Overtime)
- Experience (Years at Company, Promotions)

---

## 📊 Dashboard Features

### 🔹 Overview Dashboard
- Total Employees
- Employees Left
- Attrition Rate
- Department-wise attrition
- Attrition trend over years

### 🔹 Employee Analysis
- Attrition by Job Role
- Attrition by Age Group
- Attrition by Salary Range
- Impact of Overtime on Attrition

### 🔹 Advanced Insights
- Job Satisfaction vs Attrition
- Work-Life Balance vs Attrition
- Promotion impact on attrition
- Income vs Experience analysis

---

## 🔍 Key Insights
- Employees with low job satisfaction show higher attrition
- Poor work-life balance increases employee turnover
- Overtime significantly contributes to attrition
- Employees with fewer promotions are more likely to leave
- Mid-level experience employees show higher attrition
- Lower salary groups have higher turnover rates

---

This project demonstrates the ability to:
- Perform data cleaning and transformation using Power Query
- Create calculated measures using DAX
- Build interactive dashboards in Power BI
- Derive meaningful business insights from data
